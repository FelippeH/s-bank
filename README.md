### **Desafio de código [2]**

**Objetivo**: construir um sistema bancário com opções de login/cadastro e depósito/saque/extrato. 
A aplicação utiliza 'JSON' para armazenar e fornecer as informações quando requisitados.

## Funcionalidades:
- Acessar conta
- Criar conta
- Depósito
- Saque
- Extrato
